# The Deep Studio

**Designing Hypnotic Electronic Music**

A long-form production handbook and studio operating system for creating deep, evolving, restrained electronic music.

This repository is the master source for the book. Markdown is the source of truth. PDFs and other exports are generated from the source files.

## Project Philosophy

The Deep Studio is not a generic production guide. It is a structured handbook about attention, movement, sound design, studio architecture, and the development of a personal sonic language.

The book is built around a hybrid Ableton Live and hardware workflow using:

- Ableton Live 12 Suite
- Granulator III
- Arturia Fragments
- Elektron Analog Rytm MkII
- Elektron Analog Heat +FX
- OTO BOUM
- Boss RE-202
- Lexicon MX400
- Waldorf 2-Pole
- Mackie CR1604
- Focusrite Scarlett 18i20
- Behringer ADA8200
- Erica Synths DB-01
- Roland JU-06A
