# The Deep Studio — Release 0.1.0

## Overview

This is the first project milestone.

It establishes the repository, publishing workflow, Chapter 1 prototype,
diagram framework and editorial process.

## Included

- Repository architecture
- Front matter
- Chapter 1 manuscript prototype
- Signature diagram framework
- Publishing pipeline
- Typesetting scaffolding
- Editorial workflow

## Known limitations

- Markdown rendering is intentionally minimal.
- SVG diagrams are not yet embedded automatically.
- Chapter 1 remains a draft and will continue to evolve.

## Next release

Version 0.2.0 will focus on turning Chapter 1 into a publication-quality benchmark and strengthening the build system.
