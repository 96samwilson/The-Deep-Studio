# Style Guide

## Voice

Calm, precise, reflective and technically literate.

The book should feel closer to a thoughtful studio manual than a blog post.

## Rules

- Use British English.
- Explain **why** before **how**.
- Avoid filler, hype and listicle language.
- Avoid pretending certainty where there is only taste or preference.
- Use concrete examples from the reference studio.
- Prefer musical purpose over feature descriptions.
- Avoid generic production advice unless it is tied to the book's framework.
- Every chapter should contain both conceptual and practical material.

## Chapter Structure

Each chapter should usually contain:

1. Opening idea
2. Theory
3. Studio application
4. Practical workflow
5. Listening exercise
6. Studio mission
7. Common mistakes
8. Summary
9. Cross-references
