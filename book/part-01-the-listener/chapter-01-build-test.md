# Chapter 1 — Build Test Elements

This file exists to test the visual publishing system.

## Rendered Table Test

| Dimension | Meaning | Studio Response |
|---|---|---|
| Weight | Perceived mass, density and solidity | Saturation, compression, low-mid control |
| Distance | Perceived space and depth | Reverb, delay, darker tone |
| Movement | The feeling that sound is alive | Modulation, drift, automation |
| Focus | Where attention currently rests | Muting, arrangement, EQ priority |

## Callout Test

:::key-idea
The producer's true medium is attention through time.
:::

:::studio-mission
Create a ten-minute recording from one sound. Do not add another instrument.
:::

:::listening-exercise
Listen to a reference track and map where your attention moves.
:::

:::avoid
Do not add another layer before asking what role is missing.
:::
