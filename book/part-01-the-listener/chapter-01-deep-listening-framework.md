# Chapter 1 — The Deep Listening Framework

## Introduction

The purpose of the Deep Listening Framework is to give producers a repeatable way
to analyse music without immediately thinking about plug-ins, synthesisers or
mixing techniques.

Before asking *how* a sound was made, ask *what it is doing to the listener*.

## The Eight Listening Dimensions

The first four dimensions define the core language of this book.

| Dimension | Question |
| --- | --- |
| Weight | Does the sound feel grounded or fragile? |
| Distance | Where does it appear to exist? |
| Movement | Does it evolve or remain static? |
| Focus | Is it leading or supporting attention? |

The second four dimensions deepen the analysis.

| Dimension | Question |
| --- | --- |
| Density | How much information exists at this moment? |
| Contrast | What has changed relative to the previous moment? |
| Memory | Which sounds leave an impression after they disappear? |
| Expectation | What does the listener believe will happen next? |

## Applying the Framework

Choose a favourite track and divide it into one‑minute sections.

For each section, score every listening dimension from 1–5.

Do not write down the equipment used.

Instead, describe the experience created.

Patterns will begin to emerge that are independent of genre or technology.

## Studio Worksheet

1. Listen once without taking notes.
2. Listen again and score the eight dimensions.
3. Identify one dimension that could be strengthened in your own music.
4. Rework one existing project using only that single objective.

## Closing Reflection

The framework is not intended to replace intuition.

It exists to provide a shared language that makes intuition easier to discuss,
teach and refine.
